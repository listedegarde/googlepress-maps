== Description ==
Google Maps plugin for creating collaborative maps.  Choose the user role level for each feature.  This plugin is neither associated with Google nor Wordpress, I simply created in using their technologies.

Created by Jonathan Frazer

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `googlepress-maps` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
